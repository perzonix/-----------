# Expense Tracker 💰

Консольное приложение на Python для управления ежедневными расходами. Проект реализован в рамках обучения (Модуль II: Стандартные библиотеки и файловые операции).

## 🛠 Технологии и навыки

* **Python 3.x:** основной язык разработки.
* **JSON:** хранение и структурирование данных в локальном файле.
* **Datetime:** работа с временными метками и валидация дат.
* **Error Handling:** обработка исключений при вводе данных и работе с файлами (try-except).

## 🚀 Функционал

* Просмотр всех записей о расходах.
* Добавление новых трат с указанием суммы, категории и даты.
* Фильтрация данных по конкретной дате или категории.
* Удаление выбранных записей из базы.
* Автоматическое сохранение изменений в JSON-файл.

## 📂 Как запустить

1. Склонируйте репозиторий.
2. Запустите файл: `python main.py` (убедитесь, что скрипт имеет доступ к созданию файла `expenses.json` в той же папке).
